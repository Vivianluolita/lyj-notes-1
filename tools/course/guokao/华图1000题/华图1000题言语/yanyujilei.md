- 词语
```
世俗喧嚣”为常见搭配，喧嚣指声音杂乱，不清静，有叫嚣、喧嚷之意。
喧哗形容声音大而杂乱
福音：有利于众人的好消息
福祉：幸福美满
承受：强调忍受
承载：强调载重
设施联通，贸易畅通，资金融通、
观照”指仔细观察，审视地思考比较，主语一般是人。注意勿与表示关心照顾关怀的“关照”混淆。
和衷共济”比喻同心协力，共同克服困难
转变是侧重思想和态度的变化。所以用改变人生道路而不是用转变。
```

```
千帆竞发:是数不尽的船只竞相出发，形容事物蓬勃向上，生机勃勃地向前发展
龙争虎斗:比喻强者之间的争斗
兼收并蓄:指把不同内容、不同性质的东西收下来，保存起来
万夫不当:上万个人也抵挡不住；形容人非常勇猛
和光同尘:指不露锋芒，与世无争的处世态度
未定之天:意思是比喻事情还没有着落或还没有决定
大刀金马:①形容豪爽，气派大。②形容说话直率锋利，不留情面。 成语出处： 《儿女英雄传》第八回：“那姑娘大马金刀的坐在上面
春风化雨:多用在人或事，比喻良好的熏陶和教育。
金瓯无缺:意为金瓯没有残缺。比喻国土完整
首鼠两端:形容瞻前顾后，迟疑不决
沉疴难起:意思是久患重病，不能下床
栉风沐雨:用风梳头，以雨洗发。形容奔波劳碌，历尽艰辛。
筚路难缕:形容创业的艰辛。
失枝脱节，汉语成语，意思是比喻因关照呼应不周而造成失误.宋·陆九渊《语录》：“要常践道，践道则精明。一不践道，便不精明，便失枝落节。”
群策群力:大家一起想办法、出力气.參加這個遊戲，大家要群策群力
洋洋大观:形容事物或景象丰富多彩，盛大壮观.
鉴往知来:审视过去的经验教训，可以推知未来的发展变化。
独占鳌头:现泛指名列第一或居于首位
海晏河清：比喻天下太平。
渊渟岳峙：比喻人的品德如渊水深沉
瑕瑜互见：比喻优点缺点都有
南辕北辙：行动和目的相反
背道相驰：彼此的方向和目的相反
披沙拣金：比喻从大量事物中挑取精华
邯郸学步：模仿别人不到位，把自己忘了。
舍本逐末，舍本求末：比喻做事不抓根本，只顾细枝末节。
害马之群：比喻危害社会的人

才短思涩，汉语成语，意思是见识短浅，思路迟钝。指写作能力差.
博采众长:广泛采纳、吸取众人的长处
取长补短：通过自己的长处掩盖自己的短处
进退失据，意思是前进和后退都失去了依据；形容无处容身；也指进退两难。
沉思翰藻:寓意深刻，文辞华丽
```
